# pi.mhkb

Introducing pi.mhkb, a completely custom made HHKB style mechanical keyboard

---
# Features

1. Full HHKB layout with custom key mapping
2. Custom PCB designed in KiCAD
3. 3D printed case designed in Fusion 360
